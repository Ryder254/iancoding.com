In this chapter I Ian Muindi will study the different formats of a table i.e background, margins and spans

HTML tables allows the developers to arrange data into rows and columns
A table in HTML consists of rows and columns

Table cells

Each table cell is defined by a td and /td tags

td means table data

Everything between td and /td is the content of a table

Table rows

Each table row starts with a tr tag and ends with a /tr tag

tr Stand for table row

You can have as many table rows as possible, just make sure that the number of cells are the same in each row

Table headers

A table header is like a table data exept that it is central placed by default and bold a well

th means table header

HTML TABLE TAGS
TAG DESCRIPTION
table Defines a table
th Defines a header cell in a table
tr Defines a row in a table
tdDefines a cell in a table
caption Defines a table caption
colgroup Specify a group of one or more columns in a table for formating
col Specifies column within a <.colgroup.> elements
thead Groups the header content in a table
tbody Groups the header content in a table
tfoot Group the footer content in a table

HTML colgroup

The colgroup element is used to style specific columns in a table

You must include a colgroup element just beneath the table element if the caption is not used
There are only a very limited properties selection of CSS allowed in colgroup.

They are:

  • Width
  • Visibility
  • background
  • border
